pub struct MutableBitmap { /* fields omitted */ }
Expand description

A container to store booleans. MutableBitmap is semantically equivalent to Vec<bool>, but each value is stored as a single bit, thereby achieving a compression of 8x. This container is the counterpart of Vec for boolean values. MutableBitmap can be converted to a Bitmap at O(1). The main difference against Vec<bool> is that a bitmap cannot be represented as &[bool].

Implementation

This container is backed by Vec<u8>.
