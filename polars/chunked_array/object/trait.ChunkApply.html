Expand description

Fastest way to do elementwise operations on a ChunkedArray when the operation is cheaper than branching due to null checking

Null values remain null.

Example

fn double(ca: &UInt32Chunked) -> UInt32Chunked {
    ca.apply(|v| v * 2)
